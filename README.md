🏥 Hospital Data Analysis Project
📝 Overview
This project is a Healthcare Data Analysis Dashboard built using Power BI.
It focuses on analyzing hospital data to improve decision-making, patient management, and overall healthcare performance.
🎯 Objectives
Analyze hospital performance and patient data
Monitor patient admissions and discharges
Evaluate doctor performance and department efficiency
Identify trends in diseases and hospital utilization
Support data-driven healthcare decisions
🛠️ Tools Used
Power BI
Excel (Data Cleaning & Preparation)
Data Modeling
DAX Measures (if applicable)
📊 Dashboard Features
Interactive filters (Department, Gender, Date, etc.)
KPI cards (Total Patients, Admissions, Discharges, etc.)
Charts for patient distribution and trends
Doctor and department performance analysis
User-friendly and interactive visuals
🔍 Key Insights
Patient admissions vary across departments and time periods
Some departments have higher workload than others
Peak times for hospital visits can be identified
Data helps improve resource allocation and planning
📁 Files Included
Power BI Dashboard (.pbix)
Dataset (Excel file)
README file
🚀 Conclusion
This project highlights how data analysis in healthcare can improve hospital efficiency, patient care, and decision-making through clear and interactive visualizations using Power BI.
